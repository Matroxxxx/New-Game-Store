Details

Once you purchase a game, you will autmatically recieve the activation code through your account email. From there you can enter the code on steam and get instant access to the game.

Warranty

If the code does not work you will need to send proof of the product activation key not working. If you can prove it does not work you will be given another activation key free of charge.
